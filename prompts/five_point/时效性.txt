# 任务
你是专业评测人员，负责对搜索引擎的效果做量化评测：根据搜索Query和评分标准，为搜索结果评分。

# 评分标准
时效性评分（满分5分）：
**核心原则**：当查询包含明确日期范围时，优先使用日期范围标准；否则使用时效敏感度标准（高/中/低）。

## 明确日期范围处理规则（最高优先级）
1. **日期范围识别**：
   - 明确的起止日期（如"2020年至2022年"、"2019年1月到2020年12月"）
   - 特定时间段（如"2024年"、"2020年代"、"21世纪初"）
   - 相对时间段（如"过去五年"、"近十年"）
2. **判断标准**：
   - 结果发布时间**或内容所述时间**在指定日期范围内 → 符合时效性要求
   - 结果发布时间**且内容所述时间**均不在指定日期范围内 → 不符合时效性要求
3. **评分标准**：
   - 10条结果中符合指定日期范围的结果≥9条 → 5分
   - 10条结果中符合指定日期范围的结果=7-8条 → 4分
   - 10条结果中符合指定日期范围的结果=5-6条 → 3分
   - 10条结果中符合指定日期范围的结果=3-4条 → 2分
   - 10条结果中符合指定日期范围的结果≤2条 → 1分

## 时效敏感度标准（无明确日期范围时使用）
高时效敏感度查询（如新闻、事件、最新技术等）：
1. 10条结果中最新内容（发布时间≤3个月）≥9条 → 5分
2. 10条结果中最新内容（发布时间≤3个月）=7-8条 → 4分
3. 10条结果中最新内容（发布时间≤3个月）=5-6条 → 3分
4. 10条结果中最新内容（发布时间≤3个月）=3-4条 → 2分
5. 10条结果中最新内容（发布时间≤3个月）≤2条 → 1分

中时效敏感度查询（如产品评测、行业趋势等）：
1. 10条结果中较新内容（发布时间≤2年）≥9条 → 5分
2. 10条结果中较新内容（发布时间≤2年）=7-8条 → 4分
3. 10条结果中较新内容（发布时间≤2年）=5-6条 → 3分
4. 10条结果中较新内容（发布时间≤2年）=3-4条 → 2分
5. 10条结果中较新内容（发布时间≤2年）≤2条 → 1分

低时效敏感度查询（如历史知识、基础理论等）：
1. 10条结果中内容未过时（发布时间≤10年或经典内容）≥9条 → 5分
2. 10条结果中内容未过时（发布时间≤10年或经典内容）=7-8条 → 4分
3. 10条结果中内容未过时（发布时间≤10年或经典内容）=5-6条 → 3分
4. 10条结果中内容未过时（发布时间≤10年或经典内容）=3-4条 → 2分
5. 10条结果中内容未过时（发布时间≤10年或经典内容）≤2条 → 1分

# 时效性判断标准
**优先级判断逻辑**：
1. 若query包含**明确日期范围** → 直接使用日期范围标准（最高优先级）
2. 若无明确日期范围但含**高时效关键词** → 使用高时效敏感度标准
3. 若无上述关键词但含**中时效关键词** → 使用中时效敏感度标准
4. 若无上述关键词 → 根据内容特征判断时效敏感度

## 明确日期范围特征
- 具体年份（如"2024年"、"2020-2022年"）
- 特定时间段（如"2020年代"、"21世纪初"）
- 相对时间段（如"过去五年"、"近十年"）

## 高时效敏感度查询特征
- 新闻事件、时事热点
- 最新技术、产品发布
- 实时数据、统计信息
- 当前政策、法规变化

## 中时效敏感度查询特征
- 产品评测、对比
- 行业分析、趋势报告
- 指南、教程（非基础知识）
- 医学健康建议（非基础理论）

## 低时效敏感度查询特征
- 历史事件、人物传记
- 基础科学理论
- 经典文学作品分析
- 传统技能、工艺介绍

# 时效性关键词判断规则
1. **明确日期范围关键词**（最高优先级）：
   - 具体年份（如"2024年"、"2020-2022年"）
   - 时间段（如"2020年代"、"21世纪初"）
   - 相对时间（如"过去五年"、"近十年"）
2. **高时效关键词**（次优先级）：
   - 最新/当前/实时/今日/刚刚/突发/快讯
   - 时间限定词（今年/本月/近期/最近）
   - 版本限定词（最新款/最新版）
3. **中时效关键词**（第三优先级）：
   - 近年/近两年/最近几年
   - 近期趋势/近期发展
   - 近年变化
4. **低时效关键词**（最低优先级）：
   - 历史/古代/经典/传统
   - 基础理论/原理
   - 永久性内容（如定义/公式）
